Merge Files
Data → Merge Files
▼
Combine two or more SPSS files — either adding more cases (rows) or more variables (columns).
  1. Go to Data → Merge Files
  2. Choose Add Cases (same variables, more rows) OR Add Variables (same rows, more columns)
  3. Select the external file to merge with
  4. Match variables as needed → Click OK
💡

When to use: You collected data in two batches (same form, different months). Use Add Cases to combine them into one dataset.

↔️
Transpose
Data → Transpose
▼
Flip the dataset: rows become columns and columns become rows. Converts variables to cases and vice versa.
  1. Go to Data → Transpose
  2. Move variables to the Variable(s) box
  3. Optionally specify a Name Variable (for labeling transposed rows)
  4. Click OK — a new dataset is created
💡

When to use: Your data was collected in "wide" format (each time point as a column) but an analysis needs "long" format (each time point as a row).

🩹
Replace Missing Values
Transform → Replace Missing
▼
Estimate and fill in missing values using several methods such as Series Mean, Linear Interpolation, etc.
  1. Go to Transform → Replace Missing Values
  2. Select the variable(s) with missing data
  3. Choose a Method: Series Mean, Mean of nearby points, Median, Linear interpolation, etc.
  4. SPSS creates a new variable with the filled values (e.g., age_1)
↕️
Sort Cases
Data → Sort Cases
▼
Arrange all rows in ascending or descending order based on the values of one or more key variables.
  1. Go to Data → Sort Cases
  2. Move the key variable (e.g., age) to the Sort by box
  3. Choose Ascending (smallest to largest) or Descending
  4. Click OK — the data is now sorted
💡

You can sort by multiple variables at once. SPSS sorts by the first key, then breaks ties using the second key, and so on.

🔄
Recode Variables
Transform → Recode
▼
Convert a quantitative variable into a qualitative (categorical) one by grouping its values into ranges or categories.
  1. Go to Transform → Recode into Same Variables (overwrites) OR Recode into Different Variables (creates new)
  2. Move the variable to recode (e.g., age)
  3. Click Old and New Values
  4. Define ranges: e.g., 21–30 → 1, 31–40 → 2, 41+ → 3
  5. Click Continue → OK
🎯
Select Cases
Data → Select Cases
▼
Restrict analysis to a subset of cases based on a condition, a random sample, a time range, or a filter variable.
  1. Go to Data → Select Cases
  2. Choose selection method:
    — If condition is satisfied (e.g., Gender = 1)
    — Random sample of cases (e.g., 5%)
    — Based on time or case range (e.g., cases 100–130)
  3. Choose output: filter, copy to new dataset, or delete unselected
  4. Click OK
🧮
Compute Variable
Transform → Compute Variable
▼
Create a new variable by applying a mathematical formula to existing variables. E.g., average of several Likert scale items.
  1. Go to Transform → Compute Variable
  2. Type a name in Target Variable (e.g., mean_score)
  3. Type the formula in Numeric Expression (e.g., (x1 + x2 + x3) / 3)
  4. You can use built-in functions: MEAN, SUM, LN, SQRT, etc.
  5. Click OK — the new variable column appears in the data
💡

Syntax shortcut: For many variables, use the Syntax Editor (File → New → Syntax) to write COMPUTE commands in batch, then press Run.

📈
Create Time Series
Transform → Create Time Series
▼
Generate new variables derived from time series: differences, moving averages, cumulative sums, lags, leads, etc.
  1. Go to Transform → Create Time Series
  2. Select the time-series variable(s)
  3. Choose a Function:
    — Difference (removes trends)
    — Seasonal Difference
    — Centered/Prior Moving Average
    — Cumulative Sum
    — Lag (shift back n periods)
    — Lead (shift forward n periods)
  4. Click OK — new variables are added
🔍
Find Cases / Variables
Edit → Go To Case / Variable
▼
Navigate quickly to a specific case number or variable name in large datasets without scrolling.
  1. Go to Edit → Go To Case or Go To Variable
  2. For cases: type the row number (e.g., 100) → click Go
  3. For variables: select from the dropdown → click Go
  4. The cursor jumps to that position in the Data View
✏️
Syntax Editor
File → New → Syntax
▼
Write SPSS command scripts to automate multiple operations at once. Essential for complex computations across many variables.
  1. Go to File → New → Syntax
  2. Type your COMPUTE commands (e.g., COMPUTE y1=(x1+x2+x3+x4)/4.)
  3. Press Run → All (or Ctrl+A then Run)
  4. The computed variables appear in your data file
compute y1=(x11+x12+x13+x14)/4.
compute y2=(x21+x22+x23)/3.
compute total=(y1+y2)/2.
EXECUTE.

Merge Files — Deep Dive

Merging files in SPSS means combining two datasets into one. There are two scenarios:
(1) Add Cases — both files have the same variables, but different observations (rows).
(2) Add Variables — both files have the same cases, but different variables (columns).

SECTION 03
Replacing Missing Values

Missing values (shown as a dot "." in SPSS) can cause problems in analysis. SPSS can estimate and fill them using several strategies.

❌ Before: Raw Data with Missing Values
✅ After: Missing Values Replaced
MethodHow It WorksBest For
Series MeanReplaces with the overall mean of the variableStable, non-trended data
Mean of Nearby PointsAverage of the n closest non-missing valuesLocally stable data
Median of Nearby PointsMedian of nearby values (robust to outliers)Skewed or outlier-prone data
Linear InterpolationDraws a straight line between neighboring valuesLinearly trending time series
Linear Trend at PointFits a regression trend line and uses itStrong linear trends

SECTION 06
Select Cases

Sometimes you only want to run analysis on a subset of your data. The Select Cases function lets you filter data without permanently deleting the excluded rows.

Selection TypeDescriptionExample
If condition is satisfiedFilter by a logical condition on any variableAnalyze only Female patients: Gender = 2
Random sampleRandomly select a % of all casesSelect 5% for spot-checking
Based on time or case rangeSelect consecutive rows from N to MCases 100 to 130 only
Use filter variableA 0/1 variable indicates which cases to includeInclude case if valid_response = 1
⚠️

Important: Unselected cases are NOT deleted — they are filtered out (a diagonal line appears on their row number). You can restore all cases by selecting "All Cases" again. OR you can permanently delete them or copy selected to a new dataset.
