SECTION 01

Introduction to Exploratory Data Analysis

There has been a huge leap in the use of exploratory data analysis (EDA) tools, and their importance has increased with the rapid development in computing and the availability of software that enables their easy use. Some EDA tools serve as a preliminary step to create frequency distributions, others as alternatives, and still others can be used in advanced stages of research.

Why Use EDA? Six Key Purposes

Recognising the range of data and the pattern of distribution of observations — just by looking.
Knowing where data is concentrated (areas of high frequency).
Knowing the direction of skewness, and thus the general level of the data.
Discovering outliers and extreme values in the data.
Identifying common (frequent) values in the data.
Knowing the number of peaks in the distribution.

SECTION 02

Bar Charts & Pie Charts

Bar Chart for a Categorical Variable

A simple bar chart displays the count (or percentage) of cases in each category. The example below shows the education level distribution of 59 nurses.

Bar Chart: Education Level — N of Cases (Graphs → Bar → Simple → Summaries for Groups of Cases)

SPSS Steps: Create a Simple Bar Chart

  1. Go to Graphs → Bar.
  2. Select Simple and choose Summaries for groups of cases → click Define.
  3. In Bars Represent, choose N of cases (count) or Other statistic (e.g., mean) and move the variable to the Variable box.
  4. Move the categorical variable (e.g., Education) to the Category Axis box.
  5. Click OK. To add labels, double-click the chart and use Elements → Show Data Labels.

Pie Chart for Categorical Data

A pie chart shows proportional composition. From the lecture: marital status of 60 employees — single 33.33%, married 43.33%, divorced 13.33%, widowed 10%.

Pie Chart: Marital Status — Graphs → Pie → Summaries for groups of cases
💡

Key difference: Use a Bar chart when you want to compare counts/values across categories. Use a Pie chart when you want to emphasise each category's share of the whole (best with 2–5 categories).

SECTION 03

Histogram & Dot Plot

A Histogram is used for quantitative (numeric) variables. Bars represent frequencies in equal-width intervals. You can overlay a normal curve to visually test how well the data fits a normal distribution.

Histogram: Current Salary — Mean = 4698, SD = 1197.87, N = 60

Paneling a Chart

A paneled chart is a grid of sub-charts — same chart type and axes, but for different groups of a categorical variable. Example: one histogram for males, one for females side by side. Paneling helps compare data patterns across groups.

  1. Select options for the chart and click Define.
  2. In the dialog, move one or more categorical variables to the Panel By → Rows or Columns boxes.
  3. Check Nest variables (no empty rows/columns) if the panel variables are nested — a panel is created only for each nesting, not every combination.
  4. Click OK — SPSS produces a paneled chart grid.

Dot Plot

A Dot Plot displays the range of values on the horizontal axis and the number of observations for each value on the vertical axis — by stacking one dot per case. It reveals the exact shape of the distribution and shows where data is concentrated.

🖥️

SPSS: Go to Graphs → Scatter/Dot → Simple Dot → Define. Move the numeric variable (e.g., Experience) to the X-Axis Variable box → OK.

Dot Plot: Years of Experience — each circle = one employee, stacked by value

Stem-and-Leaf Display

The Stem-and-Leaf display is one of the modern methods used as an alternative to the histogram. It can be considered a natural image of the histogram that allows the researcher to study the pattern or behaviour of data while retaining the actual data values. This makes it more efficient than a histogram for exploring key features, and it can be used as both a tabular and a graphical method.

SECTION 05

Box Plot (Box-and-Whisker)

A Box Plot is a graphical display that helps us understand the shape and spread of a dataset. It relies on five key statistics: Minimum, Q1, Median, Q3, Maximum. It is especially useful for comparing groups in terms of spread, median, quartiles, and outliers.

How to Draw a Box Plot

  1. Set an appropriate scale on the vertical axis from smallest to largest value.
  2. Mark the positions of Minimum, Q1, Median, Q3, Maximum on the axis.
  3. Draw a rectangle (box) between Q1 and Q3. The median line appears inside the box.
  4. Draw a whisker line from Minimum up to the lower edge of the box.
  5. Draw a whisker line from Maximum down to the upper edge of the box.
  6. If there are outliers, they appear as individual points above or below the whiskers.

SECTION 06

Error Bar Plots

An Error Bar Plot shows the mean of a numeric variable for each group, plus error bars that represent the uncertainty around that mean. The error bars can represent three different things — you choose which one tells the story you need.

Error Bar TypeWhat It ShowsWhen to UseSPSS Setting
Confidence Interval (CI)The range within which the true population mean likely falls (default: 95%)Comparing group means — overlapping CIs suggest no significant differenceBars Represent → Confidence interval for mean
Standard Deviation (SD)The spread of individual values around the mean (e.g., ±2 SD)Showing natural variability within each groupBars Represent → Standard deviation, multiplier = 2
Standard Error (SE)How precisely the sample mean estimates the population meanPrecision of estimates; smaller SE = better estimateBars Represent → Standard error of mean, multiplier = 2

SECTION 07

Scatter Diagram

A Scatter Diagram is a graphical method for displaying the relationship between two quantitative variables. One variable is plotted on the horizontal X-axis, the other on the vertical Y-axis. Each observation becomes a single point on the plot. The overall pattern of points reveals the type, direction, and strength of the relationship.

Scatter Diagram — click to explore different relationships

SECTION 08

Explore in SPSS

The Analyze → Descriptive Statistics → Explore procedure is a powerful all-in-one command that produces a comprehensive summary: descriptive statistics table, stem-and-leaf display, box plot, and normality tests — all at once, optionally split by a grouping (factor) variable.

What Explore Produces

OutputWhat You SeeUse It For
Descriptives TableMean, 95% CI, 5% Trimmed Mean, Median, Variance, SD, Min, Max, Range, IQR, Skewness, KurtosisFull summary — compare across groups instantly
Stem-and-LeafText-based display retaining all actual valuesShape, outliers, common values
Box PlotGraphical 5-number summary with outlier markersSpread, skewness, outlier detection
Normality TestsKolmogorov-Smirnov and Shapiro-Wilk tests + P-P/Q-Q plotsFormally test whether data is normally distributed

SPSS Steps: Run Explore

  1. Go to Analyze → Descriptive Statistics → Explore.
  2. Move the outcome variable(s) (e.g., patient satisfaction) to the Dependent List.
  3. Move a grouping variable (e.g., gender) to the Factor List — Explore will split all output by group.
  4. In Display, choose Both (Statistics + Plots), Statistics only, or Plots only.
  5. Click Plots… to add normality plots or change box plot options.
  6. Click OK — the Output Viewer shows the full Explore report.
