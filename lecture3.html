SECTION 01

Measures of Central Tendency

A measure of central tendency determines the centre of a dataset. It is used to describe and analyse data and to infer population characteristics from sample characteristics.

x̄
Arithmetic Mean
الوسط الحسابي

The sum of all values divided by the number of observations. The equilibrium point where the sum of all deviations equals zero.

x̄ = Σxᵢ / n
  • Uses all values; unique result
  • Easy to calculate; algebraically tractable
  • Σ(xᵢ − x̄) = 0 always
  • Heavily affected by outliers
  • Poor for skewed distributions
Quantitative data only
Md
Median
الوسيط

The middle value when data is sorted in order of magnitude. Equivalent to the 50th percentile. Position: (n+1)·50/100.

Position = (n+1) × 0.50
  • Not affected by outliers
  • Works with ordinal data
  • Always exists; easy to identify
Ordinal & Quantitative
Mo
Mode
المنوال

The most frequently occurring value — the value with the highest frequency. A dataset may have zero, one, or multiple modes.

Mo = value with highest freq.
  • Not affected by outliers
  • Works for qualitative data (nominal/ordinal)
  • Easy to identify visually
Nominal · Ordinal · Scale
📊

Worked Example (n = 20 salespeople)
Data (sorted): 6, 9, 10, 12, 13, 14, 14, 15, 16, 16, 16, 17, 17, 18, 18, 19, 20, 21, 22, 24
Mean = 317 / 20 = 15.85  |  Median = position 10.5 → (16 + 16)/2 = 16  |  Mode = 16 (appears 3 times)

Weighted Arithmetic Mean الوسط الحسابي الموزون

When some values are more significant than others, we assign weights W₁, W₂, …, Wₖ.
The weighted mean is: x̄w = (W₁X₁ + W₂X₂ + … + WₖXₖ) / (W₁ + W₂ + … + Wₖ)

🎓

Example: Final exam weight = 3 × oral exam weight. Student scores: Final = 85, Oral1 = 70, Oral2 = 90.
x̄w = [(1)(70) + (1)(90) + (3)(85)] / (1+1+3) = (70 + 90 + 255) / 5 = 415 / 5 = 83

SECTION 02

Measures of Variability

Measures of variability (dispersion) describe how spread out the data values are. Two datasets can have the same mean but very different variability.

MeasureArabicFormulaKey Property
Rangeالمدى Max − Min Simplest; highly sensitive to outliers
Interquartile Rangeالمدى الربيعي Q3 − Q1 Middle 50% spread; robust to outliers
Variance (s²)التباين Σ(xᵢ−x̄)² / (n−1) Average squared deviation from mean
Standard Deviation (s)الانحراف المعياري √s² Same units as data; most widely used
Coefficient of Variationمعامل الاختلاف (s / x̄) × 100 Unit-free; compares different-unit groups

Sample Variance & Standard Deviation

s² = Σ(xᵢ − x̄)² / (n − 1)
Sample variance uses (n − 1) in the denominator (Bessel's correction)
s = √s²
Standard deviation is the square root of variance — expressed in original units
📊

Worked Example (n = 20): x̄ = 15.85  |  Σ(xᵢ − x̄)² = 378.55
s² = 378.55 / 19 = 19.92  |  s = √19.92 = 4.46
Range = 24 − 6 = 18  |  IQR = Q3 − Q1 = 18.75 − 13.25 = 5.5

Coefficient of Variation (CV) معامل الاختلاف

Standard deviation depends on units, so it cannot directly compare groups measured in different units. The CV expresses the SD as a percentage of the mean, making it unit-free.

CV = (s / x̄) × 100

The group with a lower CV is more homogeneous (less variable).

SECTION 03

Normal Curve Rule & Z-Score

The normal distribution is a symmetric, bell-shaped curve. When data follows this shape, a fixed percentage of observations always falls within a given number of standard deviations from the mean.

The Empirical (68-95-99) Rule قاعدة المنحنى الطبيعي

68%
μ − 1σ  to  μ + 1σ
About two-thirds of all observations lie within one standard deviation of the mean.
95%
μ − 2σ  to  μ + 2σ
Almost all typical observations lie within two standard deviations of the mean.
≈99%
μ − 3σ  to  μ + 3σ
Practically all observations fall within three standard deviations of the mean.
Interactive Normal Curve — hover the shaded regions
Showing the 68-95-99 rule. Click a region to highlight it.
💼

Worked Example: 500 workers, mean wage = 1400 EGP, SD = 150 EGP.
Workers earning between 1250–1550 (= x̄ ± 1σ): 500 × 0.68 = 340 workers
Workers earning between 1100–1700 (= x̄ ± 2σ): 500 × 0.95 = 475 workers

Standard Score (Z-Score) الدرجة المعيارية

A Z-score converts any raw value into a standardised score with mean = 0 and SD = 1. This allows comparison across different scales.

Zᵢ = (Xᵢ − X̄) / S

A positive Z means the value is above the mean; negative means below.

SECTION 04

Percentiles & Quartiles

Percentiles divide an ordered dataset into 100 equal parts. The Pth percentile is the value below which P% of observations lie.
Position of Pth percentile = (n + 1) × P / 100

Q
Quartiles
الربيعيات — 4 equal parts

Q1 = 25th percentile (lower quartile)
Q2 = 50th percentile (median)
Q3 = 75th percentile (upper quartile)

IQR = Q3 − Q1
D
Deciles
العشيرات — 10 equal parts

Split the ordered data into ten equal groups. D1 = 10th percentile, D5 = median, D9 = 90th percentile.

P = k × 10, k = 1…9
%
Percentiles
المئينات — 100 equal parts

The most granular split — 99 dividing values separate the data into 100 groups of equal count.

P = 1 … 99

How to Find a Percentile

  1. Sort the data in ascending order.
  2. Calculate the position: L = (n + 1) × P / 100
  3. If L is a whole number, the percentile is the value at position L.
  4. If L is fractional, interpolate between the values at positions ⌊L⌋ and ⌈L⌉ using the decimal fraction.
📋

Worked Example (n = 20, sorted data):
6, 9, 10, 12, 13, 14, 14, 15, 16, 16, 16, 17, 17, 18, 18, 19, 20, 21, 22, 24

50th percentile (median): Position = 21 × 0.50 = 10.5 → 16 + 0.5 × (16−16) = 16
80th percentile: Position = 21 × 0.80 = 16.8 → 19 + 0.8 × (20−19) = 19.8
90th percentile: Position = 21 × 0.90 = 18.9 → 21 + 0.9 × (22−21) = 21.9
Q1 (25th): Position = 21 × 0.25 = 5.25 → 13 + 0.25 × (14−13) = 13.25
Q3 (75th): Position = 21 × 0.75 = 15.75 → 18 + 0.75 × (19−18) = 18.75

SECTION 05

Distribution Shape: Skewness & Kurtosis

Beyond central tendency and spread, we describe the shape of a distribution using skewness (symmetry) and kurtosis (peakedness).

Skewness معامل الالتواء

Skewness measures the asymmetry of a distribution. Calculated as:

ρ = (X̄ − Mode) / S   or, when the median is known:   ρ = 3(X̄ − Median) / S

Range: −3 to +3. Near zero = symmetric.

Skewness ValueShapeRelationshipReal Example
ρ < 0 (negative) Left-skewed (skewed to left) Mean < Median < Mode Test scores in an easy exam
ρ ≈ 0 (zero) Symmetric (normal) Mean = Median = Mode Heights in a population
ρ > 0 (positive) Right-skewed (skewed to right) Mode < Median < Mean Monthly salaries, income

Kurtosis معامل التفرطح

PLATY
Platykurtic
مفرطح — flat distribution

Fewer values near the mean; data spread widely. Flatter peak, lighter tails. Kurtosis < 3.

MESO
Mesokurtic
تماثل — normal shape

Moderate breadth and peak — the standard normal distribution. Kurtosis = 3 (excess kurtosis = 0).

LEPTO
Leptokurtic
مدبب — peaked distribution

More values concentrated near the mean; sharply peaked with heavy tails. Kurtosis > 3.

A symmetric (mesokurtic) distribution — Mean = Median = Mode. The normal distribution is the classic example.
