SECTION 7.1

Random Variables

A random variable, X, is a function that assigns a single, but variable, numeric value to each element of a sample space. It turns the outcomes of a random experiment into numbers we can work with mathematically.

Consider four sequential births. Each child is either a Boy (B) or Girl (G), giving 2⁴ = 16 equally likely outcomes. The random variable X = "number of girls" maps each outcome to a count from 0 to 4.

DISCRETE RANDOM VARIABLE
  • Countable number of possible values
  • Discrete gaps between successive values
  • Measurable probability at each individual value
  • Examples: # of defects, # of heads in 5 flips, # of calls per hour
CONTINUOUS RANDOM VARIABLE
  • Uncountably infinite number of values
  • Moves continuously from value to value
  • No measurable probability at a single exact point
  • Examples: height, weight, time, temperature

📋

Rules of Discrete Probability Distributions: (1) P(x) ≥ 0 for all values of x. (2) ΣP(x) = 1 over all values. These two conditions must always hold.

Cumulative Distribution Function (CDF)

The cumulative distribution function F(x) = P(X ≤ x) = Σ P(i) for all i ≤ x. It gives the probability that X takes a value at most equal to x.

💡

Useful identities: P(X > a) = 1 − F(a)  |  P(a ≤ X ≤ b) = F(b) − F(a−1)  |  P(X = x) = F(x) − F(x−1)

In the 4-births example, what is P(X ≥ 3) — the probability of at least 3 girls?
P(X ≥ 3) = P(3) + P(4) = 4/16 + 1/16 = 5/16 = 0.3125. Alternatively: 1 − F(2) = 1 − 0.6875 = 0.3125.
📐
SECTION 7.2

Expected Value & Variance

The expected value (mean) of a discrete random variable X is the probability-weighted average of all its possible values:

μ = E(X) = Σ x · P(x)   (sum over all values)

μ = E(X) = Σ x · P(x)
σ² = V(X) = E[(X − μ)²] = E(X²) − [E(X)]²
σ = SD(X) = √V(X)
Mean (μ), Variance (σ²), Standard Deviation (σ)
🎯

Shortcut for Variance: σ² = Σ x² P(x) − μ²   — often easier to compute than summing (x − μ)² directly.

SECTION 7.3

Bernoulli Random Variable

A Bernoulli trial is a single experiment with exactly two possible outcomes — success (X = 1, probability p) or failure (X = 0, probability q = 1 − p). The random variable X counting successes in one Bernoulli trial is a Bernoulli random variable.

P(X = 1) = p   |   P(X = 0) = 1 − p = q
E(X) = p   |   V(X) = p(1 − p) = pq
Bernoulli PMF — mean and variance proofs come directly from the definition
⚕️
Medicine
Model single-patient events: disease onset (success) or no disease (failure). Used in clinical trial analysis.
🏦
Finance
Logistic regression uses Bernoulli distributions to model binary outcomes like bank default or no default.
🔧
Quality Control
Each inspected item is either defective (success) or conforming (failure). The foundation for sampling schemes.
🧬
Building Block
Bernoulli is the foundation for Binomial (n Bernoulli trials), Geometric, and Negative Binomial distributions.
📌

When n = 1, the Binomial distribution reduces to the Bernoulli distribution. So Bernoulli(p) = Binomial(1, p).

📊
SECTION 7.4

The Binomial Distribution

X ~ Binomial(n, p): count of successes in n independent, identical Bernoulli trials each with success probability p. Conditions: fixed n, independent trials, constant p, only two outcomes per trial.

P(X = x) = ⁿCₓ · pˣ · q⁽ⁿ⁻ˣ⁾ = n! / [x!(n−x)!] · pˣ · (1−p)⁽ⁿ⁻ˣ⁾
μ = E(X) = np   |   σ² = V(X) = npq   |   σ = √(npq)
where x = 0, 1, 2, …, n  |  q = 1 − p

SECTION 7.5

The Poisson Distribution

X ~ Poisson(λ): counts the number of events occurring in a fixed interval of time, area, or volume, where events occur independently at a constant average rate λ. Also approximates Binomial when p ≤ 0.05 and n ≥ 20, using λ = np.

P(X = x) = (e⁻λ · λˣ) / x!    for x = 0, 1, 2, 3, …
E(X) = V(X) = λ   |   σ = √λ
e ≈ 2.71828 (Euler's number)  |  λ is both the mean and variance — a unique feature!
📞
Call Centers
Model the number of calls received per hour to know how many agents to staff. If λ = 10 calls/hour, P(X = 0) ≈ 0.000045.
🍽️
Restaurants
Model customer arrivals per day to optimize staffing, seating capacity, and food preparation schedules.
🔬
Rare Events
Model rare manufacturing defects, radioactive decay events, or mutations — when p is very small and n very large.
🚗
Traffic Flow
Count vehicle arrivals at an intersection per minute to design traffic signals and road capacity.

Example — Telephone Choices

📱

1000 telephone models available. 200 managers each pick independently and uniformly. λ = np = 200 × (1/1000) = 0.2.

P(0) = e⁻⁰·² × 0.2⁰ / 0! ≈ 0.8187  |  P(1) ≈ 0.1637  |  P(2) ≈ 0.0164  |  P(3) ≈ 0.0011

SECTION 7.6

Geometric Distribution

The Geometric distribution models the number of Bernoulli trials needed to achieve the first success. It is memoryless — previous failures do not change the probability of future success. Conditions: independent trials, constant probability p, two outcomes.

P(X = n) = (1 − p)⁽ⁿ⁻¹⁾ · p = qⁿ⁻¹ · p
E(X) = 1/p   |   V(X) = q/p² = (1−p)/p²
n = 1, 2, 3, … (the trial on which first success occurs)
🎲

Example: What is the probability of rolling a 3 for the first time on the 5th roll of a fair die?
P(X = 5) = (5/6)⁴ × (1/6) = 625/7776 ≈ 0.0804
Expected roll: E(X) = 1/p = 1/(1/6) = 6 rolls on average.

GEOMETRIC vs. BINOMIAL
  • Geometric: trials until 1st success (n is random)
  • Binomial: count successes in fixed n trials (x is random)
  • Both use independent Bernoulli trials with constant p
  • Geometric is memoryless; Binomial is not
GEOMETRIC vs. POISSON
  • Geometric: discrete trials until first success
  • Poisson: events in a continuous time/space interval
  • Geometric has clear trial structure; Poisson uses rate λ
  • Different applications, both model waiting / counting
A student passes a driving test with probability p = 0.3 each attempt. What is the expected number of attempts needed to pass?
E(X) = 1/p = 1/0.3 ≈ 3.33 attempts on average until first success.
