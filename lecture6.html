SECTION 01

What is Probability?

Probability is a quantitative measure of uncertainty — a number between 0 and 1 (or 0% and 100%) that describes the likelihood of an uncertain event occurring. The closer to 1, the more likely the event; the closer to 0, the less likely.

🎯
Objective (Classical)
Based on equally-likely outcomes or long-run relative frequency. The same for all observers. Examples: tossing a coin, rolling a die, drawing a card.
🧠
Subjective
Based on personal beliefs, experience, and judgment. Differs between individuals. Example: a student saying "I will attend tomorrow's lecture with 60% probability."
📈
Empirical
Calculated by repeating an experiment many times under identical conditions. Probability = (times event occurs) ÷ (total trials).
P(A) = m / n
m = number of outcomes favourable to A  |  n = total number of equally-likely outcomes
Example 1 — Drawing a Ball

A box contains 10 balls of the same size: 6 red and 4 white. What is the probability of drawing a red ball?

P(red) = 6 / 10 = 0.60
Example 2 — Rolling a Die

A balanced die is thrown. Find the probability that an odd number appears.

Sample space S = {1, 2, 3, 4, 5, 6}  |  Event A = {1, 3, 5}

P(A) = m / n = 3 / 6 = 0.5
Example 3 — Empirical Probability

A random sample of 200 students who sat an exam found that 10 received an "Excellent" grade. Let A = the event that a student receives Excellent.

P(A) = 10 / 200 = 0.05
Example 4 — Birth Data

In a certain year in Al-Madinah, there were 10,000 births of which 5,050 were male. Let B = the event that a newborn is male.

P(B) = 5050 / 10000 = 0.505

SECTION 02

Basic Definitions & Set Theory

An experiment is any process that leads to one of several possible outcomes. A sample space (S) is the set of all possible outcomes. An event is any collection of outcomes sharing a common characteristic.

∅
Empty Set
A set with no elements. Represents an impossible event. P(∅) = 0.
S
Universal Set
Contains all possible outcomes (the sample space). P(S) = 1.
Ā
Complement
All elements of S that are not in A. P(Ā) = 1 − P(A).
∩
Intersection (AND)
A ∩ B — elements in both A and B. "A and B both occur."
∪
Union (OR)
A ∪ B — elements in A or B or both. "At least one of A or B occurs."
⊥
Mutually Exclusive
Sets with nothing in common: A ∩ B = ∅. Cannot occur simultaneously.

Example 5 — Card Drawing

Draw one card from a standard 52-card deck (n = 52). Let A = drawing a 7.

There are 4 sevens in the deck (one per suit), so m = 4.

P(A) = 4/52 = 1/13 ≈ 0.077
Card Space — Hearts & Aces

From a 52-card deck: P(Heart) = 13/52 = 1/4  |  P(Ace) = 4/52 = 1/13

P(Heart ∩ Ace) = 1/52 (only the Ace of Hearts)

P(Heart ∪ Ace) = 13/52 + 4/52 − 1/52 = 16/52 = 4/13

SECTION 03

Basic Rules for Probability

  • R1 Range: For any event A,   0 ≤ P(A) ≤ 1
  • R2 Complement Rule:   P(Ā) = 1 − P(A)
  • R3 Intersection:   P(A ∩ B) = n(A ∩ B) / n(S)  |  If mutually exclusive: P(A ∩ B) = 0
  • R4 Addition Rule (Union):   P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
  • R5 Mutually Exclusive Union:   If P(A ∩ B) = 0, then P(A ∪ B) = P(A) + P(B)
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
The Addition Rule — avoids double-counting the intersection

SECTION 04

Conditional Probability

The conditional probability of event A given that event B has occurred is written P(A|B). It asks: "If we already know B happened, how likely is A?"

P(A|B) = P(A ∩ B) / P(B)  , where P(B) ≠ 0
This also means: P(A ∩ B) = P(A|B) · P(B) = P(B|A) · P(A)
Example — IT Project Contracts (Contingency Table)

100 IT projects were assigned to AT&T or IBM, split between Telecom and Computer categories.

AT&TIBMTotal
Telecommunication.40.10.50
Computers.20.30.50
Total.60.401.00
💡

P(IBM | Telecom) = P(IBM ∩ Telecom) / P(Telecom) = 0.10 / 0.50 = 0.20
Given that a project is in Telecommunications, there is a 20% chance IBM handles it.

SECTION 05

Independence of Events

Two events A and B are statistically independent if knowing that one has occurred does not change the probability of the other. Knowing B gives no information about A.

📌

Three equivalent conditions for independence:
   ① P(A|B) = P(A)   ② P(B|A) = P(B)   ③ P(A ∩ B) = P(A) · P(B)

Example — Ace and Heart (Cards)

From a 52-card deck: P(Ace) = 4/52 = 1/13  |  P(Heart) = 13/52 = 1/4

P(Ace|Heart) = P(Ace ∩ Heart) / P(Heart) = (1/52) / (13/52) = 1/13 = P(Ace) ✓

Since P(Ace|Heart) = P(Ace), the events Ace and Heart are independent.

P(Ace ∩ Heart) = P(Ace) × P(Heart) = (4/52) × (13/52) = 1/52 ✓

Product Rules for Multiple Independent Events

✖️

Intersection of n independent events:
P(A₁ ∩ A₂ ∩ … ∩ Aₙ) = P(A₁) · P(A₂) · … · P(Aₙ)

➕

Union of n independent events:
P(A₁ ∪ A₂ ∪ … ∪ Aₙ) = 1 − P(Ā₁) · P(Ā₂) · … · P(Āₙ)

Example — TV and Billboard Advertising

Events Television (T) and Billboard (B) are independent. P(T) = 0.04, P(B) = 0.06.

a) P(T ∩ B) = P(T) × P(B) = 0.04 × 0.06 = 0.0024

b) P(T ∪ B) = 0.04 + 0.06 − 0.0024 = 0.0976

Example — At Least One Success in 10 Trials

Ten independent trials, each with failure probability 0.90. What is the probability that at least one succeeds?

P(at least one) = 1 − (0.90)¹⁰ = 1 − 0.3487 = 0.6513

SECTION 06

The Joint Probability Table

A joint probability table is similar to a contingency table but contains probabilities instead of frequencies. The row totals and column totals are called marginal probabilities.

Example — Dollar vs. Economic Growth

Economic growth level (High / Medium / Low) and whether the dollar appreciates or depreciates.

HighMediumLowTotal
$ Appreciates0.210.200.040.45
$ Depreciates0.090.300.160.55
Total0.300.500.201.00
📌

Marginal probabilities are simply the row and column totals.
For example, P(High growth) = 0.30 and P($ Appreciates) = 0.45.

Joint probability: P($ Appreciates ∩ High) = 0.21
Conditional: P($ Appreciates | High) = 0.21 / 0.30 = 0.70
