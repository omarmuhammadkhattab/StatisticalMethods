Standard Normal Distribution Table

Values show P(Z < z) — the cumulative probability to the left of z.
Example: P(Z < 1.25) = 0.8944 → read row 1.2, column 0.05.

P(Z < z) = —

How to use: For P(Z > z), compute 1 − P(Z < z).  |  For P(a < Z < b), compute P(Z < b) − P(Z < a).  |  For negative z, use: P(Z < −z) = 1 − P(Z < z) by symmetry.
